# 🎬 Movie Dataset Analysis Project

## Overview

This project is a data analysis of a movie dataset using Python. The goal is to explore the dataset, clean and process the data, and extract insights about movie genres, including identifying the most common genres.

The analysis is performed in a Jupyter Notebook.

---

## Data Download

Run the `src/data_download.py` script to download the dataset.


## Jupyter Notebook

The Jupyter Notebook contains the full analysis including:
- Data loading and inspection
- Data cleaning and preprocessing
- Genre splitting and transformation
- Visualization of genre distribution
- Identification of the top 10 most common genres
